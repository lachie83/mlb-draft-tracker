# Security Policy

## Supported Versions
This project doesn't use versioned releases — `main` is the only supported
branch and is what gets deployed. Security fixes land as regular commits to
`main`.

## Reporting a Vulnerability
Please report security vulnerabilities using
[GitHub's private vulnerability reporting](https://github.com/lachie83/mlb-draft-tracker/security/advisories/new)
for this repository (Security tab → "Report a vulnerability"). This lets you
share details privately with the maintainer before anything is public.

Please do not open a public GitHub issue for security vulnerabilities.

## What to expect
This is a personal, actively-maintained side project rather than a project
with a dedicated security team, so please be patient — reports will be
acknowledged and investigated as soon as reasonably possible, but there's no
formal SLA.
